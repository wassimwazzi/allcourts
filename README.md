# AllCourts

AllCourts is a booking-first sports marketplace for facilities and players.

## Workspace

- `frontend/web`: Next.js website and facility dashboard
- `frontend/mobile`: Expo React Native player app
- `backend/supabase`: database schema, auth, policies, seeds
- `backend/functions`: privileged server-side workflows and webhooks
- `packages/*`: shared contracts, validation, sdk, and web UI primitives

## Product direction

The initial implementation focuses on bookings, payments, reservations, community, tournaments, and leagues. Memberships are deferred, but the architecture leaves room for them later.
